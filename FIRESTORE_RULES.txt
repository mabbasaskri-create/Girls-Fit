**Firestore Security Rules (IMPORTANT):**

1. Go to: https://console.firebase.google.com/project/girls-fit-e5735/firestore/rules

2. Replace the rules with:
```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;  // TEST MODE - Change before production!
    }
  }
}
```

3. Click "Publish"

This allows your admin panel to write products and all devices to read them.

**For production later, use:**
```
allow read: if true;  // Everyone can read products
allow write: if request.auth != null && request.auth.token.email == "m.abbas.askri@gmail.com";  // Only admin can write
```
